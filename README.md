## 活動履歴の自動更新 (GitHub Agentic Workflow)

`.github/workflows/collect-activities.md` は [GitHub Agentic Workflows](https://github.github.com/gh-aw/) 形式のワークフローです。
毎週月曜日に Copilot エージェントが GitHub・はてなブログ・Zenn・Qiita のポジティブな活動を収集して `activities.md` を更新します。

### 初回セットアップ

```sh
# gh-aw CLI 拡張をインストール
gh extension install github/gh-aw

# リポジトリを初期化（COPILOT_GITHUB_TOKEN シークレットなどを設定）
gh aw init

# .md ファイルをコンパイルして .lock.yml を生成
gh aw compile
```

詳細は [GitHub Agentic Workflows ドキュメント](https://github.github.com/gh-aw/) を参照してください。
